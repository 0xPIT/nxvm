装载规则
Load CS: 权限满足；必须是可执行段；必须非空；
Load SS: 权限满足；必须是数据段；必须可写；必须非空
Load DS/ES/FS/GS: 权限满足；如果是可执行段，必须可读；如果是数据段，没限制；如果是空描述符，没限制
Load ODS/OSS：没限制
读取规则
Read CS: 权限满足；必须是可读段；
Read SS: 权限满足；
Read DS/ES/FS/GS：权限满足；不能是空描述符
Read ODS/OSS：权限满足；如果是可执行段，必须可读；不能是空描述符
写入规则
Write CS：出错
Write SS：通过
Write DS/ES/FS/GS：权限满足；必须是数据段；必须可写；不能是空描述符
Write ODS/OSS： 必须是数据段；必须可写；不能是空描述符
